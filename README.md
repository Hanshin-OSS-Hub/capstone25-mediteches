# 약 설명 4컷 만화 생성 앱 (Medication Comic Guide)

> 복잡한 복약 지시를 **누구나 이해할 수 있는 4컷 만화**로 바꿔주는 AI 기반 웹 애플리케이션

---

## 1. 프로젝트 개요 (Overview)

**약 설명 4컷 만화 생성 앱**은  
어려운 복약 지시, 주의사항, 증상 설명 등을 **텍스트 기반 설명 + 4컷 만화 형태**로 자동 변환해주는 서비스입니다.

특히 다음과 같은 대상자를 주요 사용자로 상정합니다.

- 의학·약학 비전공자
- 어린이, 청소년
- 고령자, 보호자
- 글보다 **이미지·스토리 위주 정보**가 이해하기 편한 사용자

이 앱은 **LLM 기반 자연어 처리**와 **이미지 생성 모델**을 활용하여,  
`[약 정보 입력] → [쉬운 설명 생성] → [4컷 스토리 보드 구성] → [컷별 이미지 생성]` 흐름을 제공합니다.

본 코드는 'node.js' 를 일절 포함하고 있지 않아 따로 install 이 필요합니다.

## 2. 주요 기능 (Features)

### 2.1 복약 설명 생성

- 약 이름, 질환/증상, 복용 방법, 주의사항 등을 입력하면:
  - 전문 용어를 **일상적인 표현으로 재작성**
  - 고령자/아동도 이해 가능한 난이도로 재구성
  - “해야 할 것 / 하면 안 되는 것”을 명확히 구분하여 출력

### 2.2 4컷 만화 시나리오 생성

- 생성된 복약 설명을 바탕으로:
  - **도입 – 문제 제기 – 복약 행동 – 결과/주의**의 구조를 가진 4컷 스토리 자동 생성
  - 컷마다:
    - 장면 설명 (Scene description)
    - 등장인물/캐릭터
    - 말풍선 대사
    - 강조해야 할 포인트(주의사항, 복용 시간 등)를 포함

### 2.3 4컷 만화 이미지 생성

- 각 컷별로 이미지 생성 모델(API) 호출
  - 일관된 캐릭터/스타일 유지 (예: 심플한 일러스트, 웹툰 스타일 등)
  - 텍스트는 말풍선에 표시하거나, 컷 설명 + 텍스트 오버레이 형태로 구성
- 생성된 4컷을:
  - 한 장의 이미지로 합치기
  - 또는 컷별 개별 이미지로 다운로드 제공

### 2.4 결과 저장 및 공유

- 생성된 결과를:
  - 이미지 파일(PNG/JPEG)로 다운로드
  - 향후 확장: PDF, 인쇄용 리플렛 포맷으로 내보내기
- (선택) 사용자별 히스토리 저장 기능
  - 이전에 생성한 복약 만화 다시 열람
  - 동일 약에 대한 설명 재사용

## 3. 기술 스택 (Tech Stack)

> 실제 프로젝트 환경에 맞게 수정해서 사용하세요.

**Frontend**

- React (Vite + TypeScript)
- 상태 관리: React Query / Redux Toolkit (택 1)
- 스타일링: Tailwind CSS 또는 Styled-Components
- Axios 기반 API 통신

**Backend**

- Python FastAPI (또는 Django REST Framework)
- RESTful API 설계
- OpenAI / 기타 LLM API 연동 (텍스트 생성)
- 이미지 생성 API (예: DALL·E 등) 연동

**Infra / 기타**

- Python 3.10+
- Node.js 18+
- 패키지 관리: `pip` / `poetry` + `npm` / `pnpm`
- (선택) DB: SQLite / PostgreSQL (사용자 히스토리/로그 저장 등)
- 환경 변수 관리: `.env`
