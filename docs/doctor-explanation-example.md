# 진료 후 의사 설명 예시

진료 후 기능 테스트 시 아래 내용을 "의사가 한 설명" 또는 "처방받은 약" 칸에 붙여넣기하세요.

---

## 예시 1: 무릎 통증 (정형외과)

**의사 설명:**

```
왼쪽 무릎 MRI 결과 내측 반월상연골에 경미한 파열이 확인됩니다.
현재 2등급 수준이라 수술까지는 필요 없고, 보존적 치료로 충분합니다.
대퇴사두근 강화 운동을 꾸준히 해주시고, 무릎에 무리가 가는 쪼그려 앉기나
계단 오르내리기는 당분간 피해주세요.
DNA 주사(PDRN 주사)를 2주 간격으로 3회 맞으시면 연골 재생에 도움이 됩니다.
통증이 심할 때는 처방한 소염진통제 드시고, 2주 뒤에 다시 오세요.
```

**처방약:**

```
세레콕시브 200mg 1일 2회 식후
레바미피드 100mg 1일 3회 식후 30분
```

---

## 예시 2: 감기 (내과)

**의사 설명:**

```
목이 많이 붓고 편도에 약간 염증이 있네요.
세균성 편도염이 의심돼서 항생제 처방드릴게요.
항생제는 증상이 나아져도 5일치 다 드셔야 합니다.
찬 음식은 피하시고 따뜻한 물 자주 마시세요.
3일 후에도 38도 이상 열이 계속되면 다시 오세요.
```

**처방약:**

```
아목시실린 500mg 1일 3회 식후
타이레놀 500mg 1일 3회 식후 (통증 시)
목 스프레이 1일 4회
```

---

## 예시 3: 위장 (소화기내과)

**의사 설명:**

```
위내시경 결과 미란성 위염이 확인됩니다.
헬리코박터 감염은 음성이라 제균 치료는 필요 없습니다.
PPI 약을 4주간 드시면 대부분 호전됩니다.
맵고 짠 음식, 커피, 술은 치료 기간 동안 피해주세요.
공복에 속이 쓰리면 처방한 제산제를 추가로 드세요.
```

**처방약:**

```
에소메프라졸 40mg 1일 1회 아침 식전
알마겔 현탁액 1일 3회 식간 (속쓰림 시)
```

---

## 예시 4: 피부과

**의사 설명:**

```
접촉성 피부염입니다. 금속 알레르기가 원인인 것 같아요.
스테로이드 연고를 1주일만 얇게 바르시고,
1주일 지나면 보습제로 전환하세요.
가려워도 긁으면 안 됩니다.
목걸이나 시계 같은 금속 액세서리는 당분간 피해주세요.
```

**처방약:**

```
더모베이트 연고 1일 2회 환부 도포 (7일만)
세티리진 10mg 1일 1회 취침 전
```

---

## 사용법

1. `/postvisit` 페이지로 이동
2. "진료 후" 탭에서 위 예시 중 하나를 붙여넣기
3. "AI로 쉽게 정리하기" 클릭
4. 오른쪽에 정리된 결과 확인
5. "복약 타이머" 버튼으로 자동 연동 확인
